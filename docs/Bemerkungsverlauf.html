Administrative Bemerkungen

Aus der Einsatzerfassung kann ein Report mit den wichtigsten Einsatzdaten inkl. Bemerkungen und Historien erstellt werden, damit dieses per PDF verschickt oder ausgedruckt werden kann.

Ab der Rolle ' Einsatzerfassungskontrolle ' ist das Akkordeon zusammen mit dem Icon 'Bemerkungsverlauf' sichtbar.




In den Stammdaten unter der Auswahlliste 'Bemerkungskategorie' können die gewünschten Kategorien erstellt werden.

Verkehrsregelübertretungen | Meldung an Polizei Dateien dem Einsatz oder einem Patienten hinzufügen
